Class: Ember.ArrayProxy


Extends Ember.Array, Ember.MutableArray, Ember.Object.

An ArrayProxy wraps any other object that implements Ember.Array and/or Ember.MutableArray, forwarding all requests. ArrayProxy isn't useful by itself but you can extend it to do specialized things like transforming values, etc.

Defined in: array_proxy.js.
