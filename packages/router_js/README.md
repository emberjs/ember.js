# router.js

This package was a dependency of Ember.js and was migrated into the monorepo from https://github.com/emberjs/router.js

`router.js` was always bundled in the ember-source package and was never consumed directly as a package. This has been merged into the monorepo to prepare for the upcoming [Route Manager RFC](https://github.com/emberjs/rfcs/pull/1169) to make it easier to move things around during the implementation phase.
